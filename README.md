# FinScope — Financial Intelligence Dashboard

Live stock analysis with buy/sell signals, 30+ ratios, DCF valuation, and interactive charts.

## 🚀 Deploy FREE in 5 minutes (Streamlit Cloud)

1. Fork or upload this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click "New app" → select your repo → set main file: `app.py`
4. Click Deploy — you get a live public URL like `yourname-finscope.streamlit.app`

## 💻 Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Features
- Real-time price data for any global stock
- US stocks: AAPL, MSFT, TSLA, NVDA, AMZN
- Indian stocks: RELIANCE.NS, TCS.NS, INFY.NS, HDFCBANK.NS
- DCF Fair Value estimation
- Graham Number calculation
- Buy / Hold / Sell signal with 0-100 score
- 30+ financial ratios
- Interactive Plotly charts
- Zero cost — 100% free forever
